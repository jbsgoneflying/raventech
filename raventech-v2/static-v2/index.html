Foundation brain

layer 1 · phase 0
regime encoder
64-d learned latent
Time-series transformer over 10y of multi-asset panels. Replaces the 8-factor Gaussian HMM. Produces a regime embedding plus cluster probabilities and nearest historical days.
contrastive analogues
cross-ticker · cross-time
ANN search in a learned event-embedding space. Drops E15's same-ticker constraint — a NVDA setup can pull peer analogues from 8 tickers and 5 different years that share the same fingerprint.
conformal calibrator
guaranteed coverage
Wraps every probability the engines emit. "65% breach" finally means 65% over rolling out-of-sample windows. A live coverage tracker alerts when models drift past nominal.
path generator
conditional diffusion
Replaces the OHLC MAE proxy with regime-conditional intraday paths fit to tick history. Calibrated touch / breach / MAE distributions instead of biased close-only proxies.
learned ranker
trades from desk PnL
Offline contextual bandit on the existing trade journal. Replaces hand-set composite weights. Evaluated with doubly-robust off-policy estimation so v2 ranker improvement is bounded before live A/B.
agent committee
Claude · 5-agent mesh
Researcher → Quant → Devil's Advocate → Risk Officer → Synthesizer. MCP tools over every v2 engine. Persistent memory across sessions. Streams a live debate; flags dissent on the card.
