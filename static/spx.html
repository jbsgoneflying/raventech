Entry day
Entry day (trade entry timing)

Which weekday you hypothetically open the short iron condor. We use the SPX close on that day as the entry anchor.

  • Desk view: Monday entries often “wear” more macro headline risk; Tue/Wed can reduce event density but may miss early-week premium.
  • How it’s used: it controls which historical weeks are included in the backtest and which “bucket” is used for the recommendation.

Regime
Regime score (0–100)

A composite “risk climate” score built from trend, realized vol, stress proxies, dispersion, and macro overlay.

  • How to use: lower scores are generally more favorable for short premium; elevated scores suggest widening or standing down.
  • Components: trend/vol/stress/event/disp each run 0–1 and blend into the 0–100 score.

Dealer Gamma (Live · Weekly Friday)
Dealer Gamma Context (Live, weekly Friday)

Computed from ORATS Live strike-level gamma concentration near spot (±5%) for the weekly Friday expiry. It can change intraday.

  • Positive: more pinning/mean reversion tendency.
  • Negative: more acceleration/tail risk tendency.
  • Desk guardrail: POSITIVE → OK to run “standard” wings/size if the setup is clean.
  • Desk guardrail: NEGATIVE → treat as elevated tail overlay → widen wings, cut size, or require better credit / higher EM.

Important: this does not change breach odds or any historical calculations.
