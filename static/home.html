How We Build
EOD-Only Architecture
Every signal, every computation, every output is derived from end-of-day data. No tick-level feeds, no intraday recalculation. Results are deterministic and reproducible -- the same inputs always produce the same outputs.
Immutable Snapshots
Pipeline runs produce immutable, graded snapshots that cannot be overwritten by mid-session clicks. Every snapshot carries freshness metadata (Grade A/B/C), as-of dates per region, and completeness scores for full auditability.
Idea Generation, Not Execution
Raven Tech outputs structured trade ideas with confidence scores, regime context, and suppression flags. It never executes, never sends orders, and never overrides human judgment. The desk always decides.
Math-Grounded Invalidation
Every trade idea carries three-tier invalidation rules: underlying price (EM-based), driver reversal (source-aware), and position delta. Two-of-three triggers produce hard invalidation -- no arbitrary overrides.
Global Regime Awareness
Cross-market signals from Europe, Asia, and FX flow through lead-lag analysis, regime classification, and volatility mismatch detection before any US trade idea is generated. Global context shapes every recommendation.
Silence Is Valid
When conditions are unclear, regime is stressed, or data is insufficient, the system suppresses output rather than forcing a trade. Missing data degrades confidence, never gets coerced to zero.
